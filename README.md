# Email Automation Skill

Static landing page and agent skill for emailautomationskill.com. Works with Codex, Hermes, OpenClaw, Claude Code, Cowork, and other skill-aware agents.

Install shortcut:

```bash
npx skills add emailautomationskill
npx skills add Sequenzy/emailautomationskill --skill emailautomationskill
```

Raw skill folder: `skills/emailautomationskill`
